# Diligence Memo Writer

Write the memo only after the evidence pass. This skill should compress, not invent.

## Memo Rules

- Preserve uncertainty instead of smoothing it away.
- Start with the decision and the reasons, not the chronology of the review.
- Distinguish `facts`, `inferences`, and `open questions`.
- Pull forward only the issues that matter for underwriting.
- Keep narrative claims tied to specific evidence from the specialist reviews.

Read `references/memo-guidelines.md` for the section order.

## Required Inputs

- dataroom coverage summary
- specialist findings from commercial, product/technical, and finance/legal
- risk register
- open questions

Use `assets/investment-committee-memo-template.md` for the output.
